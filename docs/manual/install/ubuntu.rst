Installing on Ubuntu
====================

PPA on Launchpad
----------------

Packages are available for both 11.10 (Oneiric Ocelot) and 12.04
(Precise Pangolin):

.. code-block:: bash

    sudo apt-add-repository ppa:tycho-s/ppa
    sudo apt-get update
    sudo apt-get install qtile
