* atmosphere_visibility
* atmosphere_pressure

* location_city
* location_region
* location_country
* units_temperature
* units_distance
* units_pressure

* wind_speed

    * - foreground

      - ``"{location_city}: {condition_temp} Â°{units_temperature}"``
      - Display format

      - True to use metric/C, False to use imperial/F

      - ``600``
      - Update interval in seconds
